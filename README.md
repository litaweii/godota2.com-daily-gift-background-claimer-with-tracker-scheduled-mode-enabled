# GoDota2 Daily Claimer

Расширение для Chrome, которое автоматически собирает ежедневный бонус (DAILY GIFT) на [godota2.com](https://godota2.com).

Сайт засчитывает бонус только если в нике Steam присутствует «godota2.com», поэтому расширение делает полный цикл за вас:

1. Временно добавляет ` godota2.com` к вашему нику Steam.
2. Авторизуется на godota2.com через Steam OpenID (если ещё не авторизованы).
3. Открывает вкладку DAILY GIFT и нажимает OPEN DAILY.
4. Считывает сумму выигрыша и записывает её в историю.
5. Восстанавливает оригинальный ник.

Всё происходит в отдельном фоновом окне браузера — можно продолжать работать, пока идёт сбор.

## Языки / Languages

Интерфейс автоматически подстраивается под язык браузера: **русский**, **English**, **español**, **Filipino**. Для остальных языков используется английский.

The UI follows your browser language automatically: Russian, English, Spanish, and Filipino (English is the fallback for everything else).

## Установка

Расширение устанавливается в режиме разработчика (в магазине Chrome его нет):

1. Скачайте и распакуйте архив (или склонируйте папку целиком).
2. Откройте `chrome://extensions` в Chrome (или Edge: `edge://extensions`).
3. Включите **Режим разработчика** (переключатель в правом верхнем углу).
4. Нажмите **Загрузить распакованное расширение** и выберите папку с файлом `manifest.json`.
5. Закрепите иконку расширения на панели — так удобнее.

## Использование

1. Войдите в Steam в этом браузере (расширение использует вашу текущую сессию — пароль ему не нужен и оно его нигде не запрашивает).
2. Откройте попап расширения и вставьте ссылку на ваш профиль Steam, например `https://steamcommunity.com/id/ваш_ник` или `https://steamcommunity.com/profiles/7656...`.
3. Нажмите **Собрать** — расширение выполнит весь цикл и покажет результат.
4. Кнопка **Тест** собирает бонус без смены ника — пригодится, если маркер уже есть в нике или вы хотите проверить авторизацию.
5. Кнопка **Остановить** прерывает процесс; ник при этом восстанавливается автоматически.

### Автосбор по расписанию

В попапе можно включить ежедневный автосбор и выбрать время. Чтобы он сработал, браузер должен быть запущен в это время.

### История и диагностика

Попап показывает общую сумму, среднее, серию и историю последних 100 сборов. Если сумма выигрыша не определилась автоматически, нажмите **Диагностика баланса** — там будет снимок страницы с последнего сбора (какие числа и попапы были видны).

## Важные предупреждения

- **Смена ника Steam блокирует обмены примерно на 3 часа.** Каждый полный сбор меняет ник дважды (маркер → откат). Если вы активно трейдите, выбирайте время автосбора, когда обмены не нужны.
- Лимит ника Steam — 32 символа. Если ник с маркером не влезает, базовая часть временно обрезается; полный ник восстанавливается после сбора.
- Если расширение не смогло восстановить ник (закрыли браузер в середине процесса и т.п.), оно дозавершит откат при следующем запуске браузера. В крайнем случае статус подскажет, каким был оригинальный ник, — его можно вернуть вручную.
- Если Steam запросит пароль, Steam Guard или капчу, расширение покажет окно и подождёт, пока вы завершите вход вручную.
- Расширение не связано с godota2.com и Valve. Автоматизация может противоречить правилам сайта — используйте на свой страх и риск.

## Приватность

Расширение не отправляет данные никуда, кроме самих сайтов steamcommunity.com и godota2.com, которые вы и так посещаете. История сборов и настройки хранятся локально в браузере (`chrome.storage`). Пароли не считываются и не сохраняются.

## Структура проекта

| Файл | Назначение |
|---|---|
| `manifest.json` | Манифест расширения (Manifest V3) |
| `background.js` | Service worker: оркестрация всего процесса, расписание, история |
| `steam-content.js` | Content script для steamcommunity.com: смена/откат ника, подтверждение OpenID |
| `godota-content.js` | Content script для godota2.com: авторизация, клик по DAILY GIFT, чтение баланса и суммы выигрыша |
| `popup.html` / `popup.js` | Интерфейс попапа: запуск, статус, статистика, расписание |
| `_locales/` | Переводы интерфейса (en, ru, es, fil) |
